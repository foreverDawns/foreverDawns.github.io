WAMPserver服务器介绍和虚拟主机搭建

WAMPServer介绍

概念

W: windows
A: Apache
M: MySQL
P: PHP

生产环境： 提供线上给用户使用的，正常工作了的项目，所在的环境的就是生产换成

开发环境： 程序员开发的时候使用的这个环境

windows下一个集成了网页服务，数据库服务，以及后台语言运行环境的集成测试环境

安装步骤

  1. 直接打开安装包
  2. 一路默认，遇到选择安装路径的时候，尽量直接安装到c盘默认的路径
  3. 可能会出现某些同学没有默认浏览器的情况，自己找到浏览器选择一下就行
  4. 还可能出现火狐浏览器的一个提示，随便选

虚拟主机的配置

就是给每一个域名绑定一个指定的根目录，在通过指定的域名来访问服务器的时候，apche会对应的找到这域名的根目录来获取文件！

  1. 开启vhosts配置， 打开httpd.conf ，找到
    #Include conf/extra/httpd-vhosts.conf
    去掉前面的#，变成
    Include conf/extra/httpd-vhosts.conf

  2. 想要在vhosts中新增一个域名和目录绑定的节点，只需要添加如下内容


DocumentRoot “网站根目录”
ServerName 域名
ServerAlias www.域名
ErrorLog “logs/dummy-host.example.com-error.log”
CustomLog “logs/dummy-host.example.com-access.log” common
Options Indexes FollowSymLinks
AllowOverride all
Order Deny,Allow
Allow from all
